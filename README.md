# Credit Card Validator

A secure and user-friendly web application for managing and validating credit card details. Built as a group project with Python Flask, it allows users to register, add cards, check validity, and toggle dark mode for a better experience.

## Features
- User registration and login with secure password hashing
- Add, view, and delete credit cards with validation (number format, expiry date)
- Masked card numbers for privacy
- Dark/light mode toggle with persistent storage
- Developer dashboard for admin access (secured by key)
- Responsive design for mobile and desktop

## Tech Stack
- **Backend:** Python Flask
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Security:** bcrypt for passwords, UUID for IDs

## Usage
- Register or log in to your account.
- Add credit cards with validation.
- View cards with masked numbers and check expiry.
- Toggle dark mode in the navbar.
- Access developer dashboard with the secret key on the landing page.
